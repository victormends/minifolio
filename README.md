# Minifolio: A Lightweight Personal Portfolio Theme for WordPress

Minifolio is a minimalist, responsive WordPress theme built on **Bootstrap**. It is designed for creative professionals who need a clean, high-performance one-page portfolio to showcase their projects and skills.

## ✨ Features

- **Responsive Design**: Mobile-first architecture using Bootstrap 3.
- **Dynamic Sidebars**: Custom sidebar regions for Hero Banners, About Me sections, and Contact areas.
- **Project Gallery**: Automated portfolio grid leveraging WordPress native `WP_Query` and Featured Images.
- **Social Integration**: Dedicated menu locations for social profiles.
- **Optimized Assets**: Enqueued scripts and styles for jQuery, Fancybox, and Animate.css.

## 🛠️ Requirements
- WordPress 5.x or higher
- PHP 7.4+

## 📂 Structure
- `functions.php`: Theme setup, asset enqueuing, and sidebar registration.
- `index.php`: Main theme template implementing the "One-Page" layout.
- `style.css`: Theme metadata and core overrides.
- `js/` & `css/`: Production-ready assets for responsive interactions.

## 🎓 Author Note
This theme was refined and restructured to follow WordPress best practices for theme development, including proper asset management and widgetized areas.
